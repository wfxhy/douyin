#### 介绍
抖音爆店码_同城码系统

#### 软件架构

特别感谢
> 项目基础采用若依

#### 主要功能
- 数据看板：活动浏览次数，转发次数；优惠券领取次数，核销次数
- 代理商管理
- 门店管理
- 额度充值
- 优惠券管理
- 优惠券领取
- 优惠券核销
- 活动管理
- 视频管理
- 视频AI混剪
- 背景音乐合成
- 视频带评论
